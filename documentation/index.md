# Budget Buddy Docs 🛒

[!badge variant="success" text="v1.2.0"] [!badge variant="info" text="MIT License"]

Welcome to the official documentation for **Budget Buddy**. Whether you are a student trying to stretch your allowance, a savvy shopper hunting for the best deals, or a community contributor helping us keep prices updated, you are in the right place.

We rely on a community-driven approach to make sure everyone has access to accurate, up-to-date pricing across local stores.

## Jump Right In

**Just want to use the app?**
Head over to the live platform and start saving today:<br />
👉 **[Budget Buddy Iligan City](https://productprice-iligan.vercel.app/)** 👈

**Want to run it locally?**
If you are a developer looking to tinker with the MERN stack codebase, getting started is easy. Clone the repository and spin up the development server:

```bash
git clone https://github.com/KishonShrill/BudgetBuddy-IliganCity.git
cd budget-buddy
npm install
npm run dev
```

## Brief
We built this platform to solve a simple problem: finding the best grocery prices shouldn't require visiting five different stores.

- **Real-Time Price Tracking**: Instantly search for everyday items and compare prices across multiple locations (like Gaisano, Shoppe 24, etc.).
- **Community-Powered Data**: Prices are sourced, verified, and updated by shoppers like you.
- **Smart Cart & Budgeting**: Add items to your virtual cart, set your budget limit, and let the app warn you if you are going over budget.
- **Rank & Reward System**: Build trust by submitting accurate prices and voting on the Community Hub to rank up to Moderator status.
